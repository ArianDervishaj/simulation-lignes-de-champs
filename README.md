### Lancer le projet

#### 1. Commencer par compiler le projet :

```bash
make main
```


#### 2. Lancer le main avec le nombre de charges positive ($1), le nombre de charge négative ($2) et le nombre de lignes de champs à générer ($3).

```bash
./main $1 $2 $3
```
